Binary search bu algaritm.

Bu yerda men algaritmni chuntirmayman. Kod larda kerakli va asosan tashlab ketiladiganlarni ko'rsataman.

BS da kodlash 2 ta versiyaga bo'linadi. Bittasi recursive version, ikkinchisi iterative version.

BS bizga array dagi kerakli elementni topishga yordam beradi. Va javob asosan index da chiqadi. Biz elementni kiritamiz. algaritm uni tezkor uchun bilan topib uning indexini beradi.

1.iterative version => recursive dan samarali va epchil versiya => space complexity O(log N)
2.recursive version => chunish va uni bajarish oson. => space complexity O(1)

BS bizga elementning indexini topib beradi. eng tezkorlik bilan. 100 ta element ichidagi element ni 10 ta qadam bilan topadi.

agar biz index() funksiyasidan  foydalansak bu ancha, juda asta kechadi.time complexity: O(N*log n)
